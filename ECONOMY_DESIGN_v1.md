## 1. FA INCOME PER RUN
Il sistema economico si basa sui **Frammenti di Archivio (FA)**, ottenuti completando run o obiettivi.

### Calcoli Guadagno FA
Il guadagno dipende dalla difficoltà e dalla percentuale di obiettivi completati.
*Extraction Multiplier:* Successo = 100%, Fallimento = 10% (Pity bonus).

| Difficoltà | Min (Fail/Low) | Avg (Success/Med) | Max (Perfect Run) | Durata Target |
| :--- | :--- | :--- | :--- | :--- |
| **Facile (Story)** | 5 FA | 75 FA | 100 FA | 15 min |
| **Normale (Chase)** | 18 FA | 180 FA | 250 FA | 12 min |
| **Difficile (Gilda)** | 45 FA | 450 FA | 650 FA | 20 min |

**Onboarding (Run 1-5):** Bonus fisso di +50 FA a run per accelerare il primo acquisto (Torcia).

## 2. CURVA DI PROGRESSIONE
- **Early Game (Run 1-5):** Accesso agli strumenti base. Accumulo stimato: ~400 FA.
- **Mid Game (Run 10-25):** Sblocco Path avanzati (SCP, Analog) e Tier 1 Upgrades. Accumulo: ~4.000 FA.
- **End Game (Run 50-100+):** Specializzazioni Gilda e cosmetici prestigio. Accumulo: >20.000 FA.

## 5. BALANCE CHECK (F2P vs Premium)
- **F2P Access:** Tutti i contenuti di gameplay (Path, Entità, Strumenti) sono sbloccabili via FA.
- **Tempo Sblocco Tool Tier 1 (500 FA):** ~3 run Normale (~36 minuti di gioco effettivo).
- **Tempo Sblocco Slot Gilda (5.000 FA):** ~28 run Normale (giocando bene).

## 6. LIVE OPS ECONOMY
- **Weekend Boost:** 1.5x FA per le run nel weekend.
- **Guild Pool:** Le gilde possono accumulare FA in un fondo comune per sbloccare upgrade globali per i membri.
